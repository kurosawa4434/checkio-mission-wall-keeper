This is puzzle-solver mission ( like a Sudoku Solver, Crossword Solver , 8 Puzzle, Bulls and Cows, Minesweeper )

This time, You should implement solver of Lights Out.

rule of the puzzle

  • You give a 5x5 panels.( 1 : Lighting up, 0 : Lights off )
  • if you click a panel, The state of the panel and adjacent panels is reversed.
  • The goal is all panels Lights off.

Input: 5x5 panels as a list of strings.

Output: Click on panels as a list of Integers.

How it is used: Get puzzle automatic answer.

Precondition:
len(wall) = 5
all(len(row) == 5 and len(row) == len(wall[0]) for row in wall)
